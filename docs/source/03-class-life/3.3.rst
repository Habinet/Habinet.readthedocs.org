Как просмотреть объявление?
---------------------------
**Через Хабинет:**

.. image:: ../_images/03-class-life/4.png 

1. Откройте "Главное меню" и выберите пункт "Объявления".

2. Нажмите по необходимому объявлению, чтобы просмотреть полную информацию.
